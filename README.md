# Arduino-Button-Example

## Contextualization
In this experiment, an LED will be turned on by pressing a button. Please note that no ports will be used and no code will be written.

## Components:
- 1 - Arduino Uno R3
- 1 - 220 Î© Resistor
- 1 - Protoboard
- 5 - Jumpers(male-to-male)
- 1 - Button
- 1 - USB Cable 
- 1 - LED
